## プロジェクト終了

プロを目指すために利敵を回避しようというコンセプトで始まる
↓
利敵リストのような機能が出来上がる。
↓
exit_cheat_bsというプロジェクト名のAWSサーバー費用がかさんで一旦サービスを閉じる
↓
がちばでプロ行く人はなんだかんだで毎回プロにいってる、本人の実力とかメンタリティの方が大事なんじゃないかと思えて来たりする
↓
バフィーあとのブロスタの資源不足をどうにかしようとする
↓
検索されたプレイヤーがリアルタイムで流れてくるタイムラインとか作ってみる、プレイヤーのプロフィールやブロウラーのスタパ、ギア、ガジェ、バフィーとか見えるようにした
↓
XやSNS強くないので誰も検索せずタイムラインにも流れてこないし、競合サービスはブロウラーの衣装まで見えるようになってて、自分も一ユーザーとしてそっちを自然に利用するようになる
↓
safebrawlというサービス自体を閉じた(AWS, お名前.com, cloudflare worker etc)


## 閉じた理由

ブロスタのガチバトルを以前よりやらなくなった。以前は夜通しガチバトルをしていたが、そこまでの熱量でやるよりフレンドや野良とバイロンとか使って普通にプレイしてた方が楽しいと感じ始めた。
