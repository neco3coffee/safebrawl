## プロジェクト終了

プロを目指すために利敵を回避しようというコンセプトで始まる

↓

利敵リストのような機能、プレイ動画アップロード、ログイン、ポイント機能出来上がる。

↓

あまり誰もログインしないし動画アップロードとかも一人しかしなかった。discordでレビューする機能も作ったけど思うように使われない。

↓

AWSサーバー費用がかさんで一旦exit_cheat_bsというプロジェクト名のサービスを閉じる

↓

がちばでプロ行く人はなんだかんだで毎回プロにいってる、本人の実力とかメンタリティの方が大事なんじゃないかと思えて来たりする

↓

バフィーあとのブロスタの資源不足をどうにかしようとする

↓

検索されたプレイヤーがリアルタイムで流れてくるタイムラインとか作ってみる、プレイヤーのプロフィールやブロウラーのスタパ、ギア、ガジェ、バフィーとか見えるようにした

↓

XやSNS強くないので誰も検索せずタイムラインにもプレイヤーのプロフィールカードが流れてこないし、競合サービスはブロウラーのスキンまで見えるようになってて、自分も一ユーザーとしてそっちを自然に利用するようになる

↓

safebrawlというサービス自体を閉じた(AWS, お名前.com, cloudflare worker etc)


## 閉じた理由

ブロスタのガチバトルを以前よりやらなくなった。以前は夜通しガチバトルをしていたが、そこまでの熱量でやるよりフレンドや野良とバイロンとか使って普通にプレイしてた方が楽しいと感じ始めたため。作ってる自分自身が悩みから遠ざかって共感できなくなってきたためかもしれない。。。
