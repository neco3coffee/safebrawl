## 目的


### 快適にガチバトルをプレイすること

- ３vs3で一人動かなかったら勝てない
- 心が折れて利敵しちゃうときだってある
- でもやっぱプロいってみたい



## safebrawlでなにができるのか

0. ログイン
1. 利敵をした人のマイリストをつくって、直近でガチバをまわしてるか手軽にわかる
2. 現在のシーズンのプレイされたガチバマップがみれる
3. トッププレイヤーのマップごとのピック率や勝率がわかる


## 実際に実装する機能

- [ ] ホーム画面
- [ ] プレイヤー詳細画面
- [ ] ログイン,マイプロフィール画面
- [ ] マップ詳細画面

ホーム | マイプロフィール | 利敵プレイヤーリスト

プレイヤー詳細

マップ詳細


#### ホームで できること

- [ ] プレイヤーのタグを入力することでプレイヤー詳細に遷移できる
- [ ] 直近で検索されたプレイヤーが上部の右から左に流れていく、タップするとそのプレイヤー詳細に遷移できる
- [ ] 現在のシーズンのプレイされたガチバマップが見れる


#### マイプロフィールで できること

- [ ] ログイン
- [ ] 自身のプレイヤープロフィール・バトル履歴の閲覧
- [ ] バトル履歴からワンタップで利敵プレイヤーをリストに追加


#### 利敵プレイヤーリストで　できること

- [ ] リストのプレイヤーが直近でガチバトルをまわしてるか・現在のランクがわかる
- [ ] リストのプレイヤーをタップで詳細画面に遷移


#### プレイヤー詳細で できること

- [ ] プレイヤーのプロフィール・バトル履歴・使用可能キャラの閲覧


#### マップ詳細で できること

- [ ] マップの画像がみれる
- [ ] トッププレイヤーのピック率・勝率がわかる


## やらないこと

- 名前履歴を追跡すること
- 利敵プレイヤーリストを１ヶ月以上保持すること
- 利敵プレイヤーリストの公開や共有
- 操作の妨げになる大きさの広告が自動で差し込まれること
- 月の収支が -1000を下回ること



## やってみたいこと

- 自分のピックがトッププレイヤーのどの人と似てるピックなのかがわかる機能とかあったら面白そう
