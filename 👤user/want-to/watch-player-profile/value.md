## value

- フレンドじゃないプレイヤーのプロフィールを詳細にみることができる
  - どんなキャラ(ブロウラー)を使えるか？、パワー、ギア、スターパワー、ガジェットまで見れる
  - 誰と一緒にプレイしているのか？一緒にプレイしてる人のプロフィールも見れる
  - 前はどんな名前だったのか？


## 何がみたいのか？

- プレイヤー名(色も)
- プレイヤーの名前履歴
- ガチバトルの現在のランク
- ガチバトルの最高ランク
- 使えるキャラクター(ブロウラー)
  - パワー
  - ギア
  - スターパワー
  - ガジェット
  - ハイチャ(できたら)
  - バフィー(できたら)
- バトル履歴


## その他必要なプレイヤー情報は？

- アイコン
- 現在のトロフィー
- 最大トロフィー
- クラブ名(色も)
