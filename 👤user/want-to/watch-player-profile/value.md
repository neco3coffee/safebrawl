## value

- フレンドじゃないプレイヤーのプロフィールを詳細にみることができる
  - どんなキャラ(ブロウラー)を使えるか？、パワー、ギア、スターパワー、ガジェットまで見れる
  - 誰と一緒にプレイしているのか？一緒にプレイしてる人のプロフィールも見れる
  - 前はどんな名前だったのか？


## 何がみたいのか？

- プレイヤー名(色も)
- プレイヤーの名前履歴
- ガチバトルの現在のランク
- ガチバトルの最高ランク
- 使えるブロウラー
  - パワー
  - ギア
  - スターパワー
  - ガジェット
  - ハイチャ(できたら)
  - バフィー(できたら)
- バトル履歴


## その他必要なプレイヤー情報は？

- アイコン
- 現在のトロフィー
- 最大トロフィー
- クラブ名(色も)



## めも

プレイヤー基本情報
アイコン、Tag、名前、クラブ、現在のランク、トロフィー、キャラクター数(レベル、ガジェ、スタパ、ギア)などの基本情報を表示

バトル履歴
直近のバトル履歴25件を一覧表示。各バトルの詳細情報(モード、マップ、使用キャラ、勝敗、MVP)も確認可能
